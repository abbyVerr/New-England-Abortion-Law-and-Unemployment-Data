# New-England-Abortion-Law-and-Unemployment-Data
A dataset containing abortion statistics, abortion laws, and unemployment data about each state in New England from 2011-2022. Includes code used to import, clean, and combine data. Information is from CDC, KFF, and BLS.

Branches:
imported_data - includes all imported files
metadata_imported_data - contains code used to import files and data
cleaned_data - includes all cleaned files
metedata_cleaned_data - contains code used to clean files and data
scripts - includes two quarto documents, showing the importing and cleaning processes
output - includes the finished csv document
metadata_output - contains code creating the final csv document and explains each variable
